# PortafolioMIRTS

This is a personal portfolio website built with React, Vite, and Bootstrap.

## Features

- **Responsive Design**: Built with React Bootstrap for completely responsive layouts.
- **Animations**: Uses `animate.css` and `react-on-screen` for engaging scroll animations.
- **Carousel**: Project showcase using `react-multi-carousel`.
- **Contact Form**: Integrated with Mailchimp via `react-mailchimp-subscribe`.

## Components

- **NavBar**: Responsive navigation bar with active state handling.
- **Banner**: Hero section with typing text effect and welcome message.
- **Skills**: Carousel displaying technical skills.
- **Projects**: Grid layout showcasing projects with hover effects.
- **Contact**: Contact form component (requires Mailchimp configuration).
- **Footer**: Simple footer with social links.

## Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

## Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd PortafolioMIRTS
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the app for production.
- `npm run preview`: Locally preview the production build.
- `npm run lint`: Runs ESLint to check for code quality issues.
